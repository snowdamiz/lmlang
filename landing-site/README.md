# lmlang Landing Site

Standalone landing page for lmlang, built with:

- React 19
- Vite
- Tailwind CSS
- shadcn/ui-style component primitives

## Development

```bash
npm install
npm run dev
```

## Build Static Assets

```bash
npm run build
```

Production output is generated in `dist/` and can be deployed directly to static hosts like Vercel, Netlify, Cloudflare Pages, S3 + CloudFront, or GitHub Pages.
